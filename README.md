# BMI・標準体重計算のコンソールアプリ

## 概要

インターンの導入として作成。
ユーザが入力した身長と体重に応じて、BMI と標準体重を計算するコンソールアプリ。

## 主な機能

1. **BMI 計算:** 身長と体重から BMI を算出
2. **標準体重計算:** 身長から適正な体重を計算
